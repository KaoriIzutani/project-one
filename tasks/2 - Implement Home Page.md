# Task 2 - Implement Home Page

## Instructions
1) Read this issue: https://github.com/codejourneys-org/project-one/issues/7
2) Create a new issue in the project page (https://github.com/codejourneys-org/project-one/projects/1)
3) The issue name must be "Implement page - YOUR_NAME"
4) Create a new folder /src/YOUR_NAME"
5) In src/YOUR_NAME, create the following folders: css, js, html
6) Create html/homepage.html
7) Create css/homepage.css
8) Implement the home page. All the acceptance criteria must be satisfied.
9) When you open the final Pull Request, the title contain "fix ISSUE_ID". You can find the issue id in your GitHub issue page.
10) Respect the given layout but be creative
11) Fonts and colors are not important now

Dont't forget to create a feature branch. A good one could be "feature/implement-homepage-YOUR_NAME".
